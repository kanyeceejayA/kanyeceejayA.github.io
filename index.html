School Shuttle Services Management System

For my final year school project, I built a system for school shuttle tracking. It tracked the location, speed, and staff aboard each shuttle so as to avail this information to parents and administrators. Once logged in, a parent can view the details of the shuttle that his/her child is aboard.

I used mostly PHP and JS to build it, with Java being used for a simple Mobile application. An online demo of the web app is available at gps.kirungi.co.ug.
